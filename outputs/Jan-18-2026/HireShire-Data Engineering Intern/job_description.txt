About the job
About HireShire



HireShire is a global staffing and talent development platform dedicated to connecting high-potential professionals with forward-thinking organisations. As we expand our technology talent ecosystem, we are launching our Data Engineering Internship Program to support aspiring engineers in building real-world skills across modern data systems and cloud infrastructure.




Role Overview



As a Data Engineering Intern at HireShire, you will gain hands-on experience working with scalable data pipelines, cloud data platforms, and analytics infrastructure. This role is designed for motivated learners who want to develop strong foundations in data engineering while working on real-world projects under professional mentorship.


No prior professional experience is required; this program supports beginners transitioning into the data domain.




Key Responsibilities



Learn to design and build data pipelines using tools such as Python, SQL, and Apache Airflow
Assist in collecting, transforming, and loading (ETL) data from multiple sources
Support data warehousing solutions using platforms like BigQuery, Snowflake, Redshift, or Azure Synapse
Work with cloud storage systems on AWS, Azure, or GCP
Help maintain data quality, integrity, and documentation best practices
Participate in real-time projects involving analytics datasets and reporting pipelines
Collaborate with mentors and team members to troubleshoot and optimise workflows




Who Can Apply



Students or recent graduates in Computer Science, IT, Engineering, Mathematics, or related fields
Career switchers eager to enter the data engineering domain
Candidates with:
Basic knowledge of SQL or Python
Interest in data systems, cloud platforms, and automation
Strong learning mindset and problem-solving skills
Previous experience is helpful but not mandatory




Nice to Have (Not Required)

Exposure to cloud platforms (AWS, Azure, GCP)

Understanding of Big Data concepts (Spark, Kafka, Hadoop)
