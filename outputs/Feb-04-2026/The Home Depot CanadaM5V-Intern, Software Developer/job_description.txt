Job Summary:

The Askuity Fusion Demo Data system allows configurable data to support scalable demo and education activities for suppliers.  Demo data is designed to guide specific narratives within Fusion that can be shared securely with multiple suppliers concurrently.  This project is focused on the user experience for Askuity associates to configure that data.  The process is currently manual and somewhat technical and will aim to ease the data configuration process for non-technical associates.  This will be accomplished via easy-to-use front-end interfaces for data creation and editing.

This project is valuable for associates tasked with designing merchandising focused data for product demos and education across multiple suppliers concurrently.
