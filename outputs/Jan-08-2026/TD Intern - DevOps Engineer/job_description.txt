Job Description

The Enterprise DevOps pipeline helps TD get to market faster on public cloud by enabling "Self-service" environment creation, and modularization of both the application and the infrastructure through the pipeline.

You will take part working and contributing towards various development work, building up new DevOps tools to enable deployment of Azure public cloud services. Day 1, you will treated as an equal member of the team and given meaningful assignments to help us deliver improvements to our technology platforms.

This may include, but is not limited to the following:

Work on DevOps development, design, and build, managing tools for Enterprise-wide technology including Nexus binary repository, GitHub code repository, Jenkins, CI/CD engine, Terraform, Sonarqube, & Aquasec
Learn to develop and write Python, Java, shell scripts and groovy code
Learn to setup a development environment with docker, and development tools
Learn about cloud security, working with various cloud security tools, policies and secrets management
Work with DevOps and cloud tools including GitHub, Terraform, Nexus, Jenkins, Xeebia Release & Deploy, Hashicorp Vault, Aquasec, Bitbucket
Gain exposure to Salt, DNS, Redhat Satellite, Datadog, Sensu, SQL, Splunk, Mongo
Learn cloud services including Azure machine learning, Azure SQL, Azure data lake (ADLS), Azure Kubernetes, Azure function, Azure Databrick, etc.
Review and assess new technologies and work through testing, validation, and integration for TD internal tools & systems
Continuously enhance knowledge/expertise and keep current with leading-edge technologies trends/developments and develop expertise in TD services, applications, infrastructure, analytical tools and techniques that can contribute to effective solution development/delivery
